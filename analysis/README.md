# Analysis Output Directory

This directory contains generated analysis outputs from running the Oraculus audit pipeline. Outputs are generated by the various analysis engines:

- `ace/` — Anomaly Correlation Engine reports
- `jim/` — Judicial Implication Mapping reports
- `semantic/` — Semantic harmonization matrices
- `vendor/` — Vendor influence analysis
- `pdf_forensics/` — PDF metadata forensics
- `logs/` — Pipeline execution logs

To generate analysis outputs, run:

```bash
python scripts/run_full_pipeline.py --source oraculus/corpus/ --output analysis/
```
