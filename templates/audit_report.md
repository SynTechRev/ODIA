## Methodology

This audit was conducted using the Oraculus DI Auditor (ODIA) platform, which applies
the following anomaly detectors to each ingested document:

- **Fiscal Detector** — Identifies missing budget references, unverified appropriations,
  unsigned financial instruments, and provenance gaps in fiscal records.
- **Constitutional Detector** — Flags clauses that may conflict with constitutional
  protections, including surveillance overreach and rights limitations.
- **Surveillance Detector** — Detects surveillance-related procurement, data retention
  policies, and monitoring language that lacks accompanying governance controls.

All documents are ingested with SHA-256 provenance tracking. Analysis results are
deterministic given the same input text.
